# ng-blog

A full-stack blog application built with Angular and TypeScript.

## Architecture

- **Frontend**: Angular SPA for reading and writing blog posts.
- **Backend**: Node.js/Express (or similar) API.

## Setup Instructions

### Prerequisites
- Node.js installed

### Running the Backend
```bash
cd backend
npm install
npm start
```

### Running the Frontend
```bash
cd frontend
npm install
npm start
```

The frontend will be available at `http://localhost:4200`.

## State Management
This project uses Angular's built-in state management techniques (or NgRx if configured) to handle the list of blog posts and user authentication state.
